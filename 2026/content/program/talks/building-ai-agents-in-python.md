AI agents can feel abstract until you actually build one.

This session is a practical, step-by-step walkthrough of building a custom AI agent in Python using Strands Agents SDK. We’ll start from a minimal agent and progressively add the core building blocks that make agents useful: the agent loop, lifecycle, hooks, tools, memory, and conversation management.

Instead of jumping straight into complex architectures, we’ll focus on understanding how each piece works and how they fit together. By the end, you’ll have a working mental model of how agents operate and a solid foundation you can build on.
