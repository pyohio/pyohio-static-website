The Django web framework is an amazing piece of software. It's been around for 20 years and just keeps going. Join me to learn about ten features that I think are pretty cool in the Django codebase. There's only one rule, the code must be in the core part of Django.

This talk will provide an introduction to each of the features, why it's useful and when you may want to reach for it. It's more of a punch-and-move style talk where we will skip around the codebase. To fully understand any of the features, please refer to the provided resources at the end of the talk.

The ten features we'll go through will be in the following areas:

- Four in the ORM
- One in URL Routing
- One in Forms
- One in Testing
- One wild one in the Admin
- One in Migrations
- One in Documentation
