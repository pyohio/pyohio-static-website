# Code of Conduct

Everyone interacting at PyOhio events and in online spaces is expected to follow the [PSF Code of Conduct](https://www.python.org/psf/conduct/).

## Reporting

All incidents should be reported by emailing [safety@pyohio.org](mailto:safety@pyohio.org). This email alias contacts the Response Team.

For any report that involves a member of the Response Team, email [conduct-wg@python.org](mailto:conduct-wg@python.org) instead.
